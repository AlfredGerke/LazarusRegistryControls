Packages der LRC
================

In diesem Ordner werden alle notwendigen Dateien für die Packages zur Verfügung gestellt.

**Inhaltsübersicht:**    

1. registrysource.lpk
2. registrycontrols.lpk        


registrysource.lpk
------------------

Das Package *registrysource.lpk* ist das erste Package welches installiert werden 
muss, um die LRC in Lazarus zur Verfügung zu stellen.   

Folgende Sourcen werden installiert:

1. TRegistrySource
2. Erweiterung der Eigenschaften
3. Property- und Komponenteneditoren
4. Konstanten und Resourcestrings

registrycontrols.lpk
--------------------

Das Package *registrycontrols.lpk* kann erst nach dem Package *registrysource* installiert werden.
Erst mit diesem Package werden die Steuerelement zur Verfügung gestellt.

Folgende Steuerelemente werden installiert:

1. TRegEdit
2. TRegComboBox
3. TRegCheckBox
4. TRegListBox
5. TRegRadioButton 
6. TRegRadioGroup
7. TRegCheckListBox
8. TRegCheckGroup
9. TRegValueListEditor
10. TRegLabel
