Übersetzungen für die LRC
=========================

In diesem Ordner werden alle notwendigen Dateien für die Übersetzung der Resourcestrings zur Verfügung gestellt.

**Inhaltsübersicht:**    

1. *.po        


*.po
---------------

Die *`*.po-Dateien`* werden mit Ausnahme der *example.po* nur von der LRC verwendet.
Die example.po muss in den Ordner *`..\expample\Languages\de_DE\`* kopiert werden um 
z.B. eine deutsch Übersetzung der Strings für die Beispielanwendung zur Verfügung zu stellen.

Will man die Strings in der Maske von *TEditSettings* übersetzen, muss man nur der *example.po* Datei des
Beispielprogrammes die übersetzten Stringressourcen aus der Datei *dlgeditsettings.LANG_ID.po*
hinzufügen. Für eine Übersetung ins Deutsche wären das die Stringressourcen aus der 
Datei *dlgeditsettings.de.po.*
